Webix Jet Demo App
===================

### How to run

- run ```npm install```
- run ```npm start```
- open ```http://localhost:8080```

or

- run ```yarn install```
- run ```yarn start```
- open ```http://localhost:8080```

For more details, check https://www.gitbook.com/book/webix/webix-jet/details

### Other commands

#### Run lint

```
npm run lint

or

yarn lint

```

#### Build production files

```
npm run build

or

yarn build
```

After that you can copy the "codebase" folder to the production server
